Notes

  • Java

    1. Iterators

      Used to traverse through a collection using next, hasNext and remove.
      Iterator interface: implemented by all collection classes

    2. Collections

      A collection can contain multiple objects and can do simple things such as add more objects, removing objects and sorting.
      Collection interface: implemented by all collection classes
      Collections class: contains static methods to modify preexisting collections

      1. Queues

        A collection where elements are added to the top and removed from the bottom.
        Queue interface: contains base methods for queues
